Monet

Monet is a database kernel developed to experiment with implementation techniques for novel application domains on parallel and distributed processing platforms.

These Web pages contain information on Monet:

  • Papers & Reports
  • Documentation
  • Source Code Web
  • Performance Studies
  • Monet project team

The Monet database kernel is developed at CWI and UvA.
